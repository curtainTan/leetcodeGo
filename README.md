### 广度遍历

要点：
1. 需要一个队列 queue
2. 每一层遍历时，将当前长度取出来，确保遍历本层节点
3. 循环 长度 次，每次 shift() 一个节点
4. 把下层节点 posh() 到队列中

案例：
1. 树的层序遍历
2. 走迷宫
3. 寻找最短路径
